Githublink: https://github.com/SebaTheProgrammer/RetroGameEngine

Extra info:
WASD en spatie (voor keyboard)
D-Pad en 'A' voor controller
(In game moet je de knoppen inhouden om te bewegen, zo is er nog dat retro gevoel)

Er is een highscore lijst dat wordt ingeladen met files.
Je kan ook je naam veranderen.

Er zijn 3 levels met elk 3 stages.

Er is een single player mode.
Co-op waarbij er twee Qberts zijn.
Versus waarbij er 1 speler coily is (de speeler krijgt pas controle beneden)

Er zijn 3 soorten enemies:
-Coily die je volgt en van de pyramide springt als de speler op een disc staat.
-SlickSam, die cubes reversen
-UggWrongway, die de verkeerde kant opgaan.

Engine:
Er is natuurlijk een game loop met een update methode
Game objects met components
Input en commands
Observers
Sound met threading
Ect...

Als u vragen heeft, stel ze gerust.
Happy playing!
